# 示例音频文件

这个目录包含用于演示的音频样本。

## 文件结构

```
sample-audio/
├── user1/
│   ├── tap_001.wav
│   ├── tap_002.wav
│   ├── tap_003.wav
│   └── ...
├── user2/
│   ├── tap_001.wav
│   ├── tap_002.wav
│   └── ...
└── noise_samples/
    ├── ambient_noise.wav
    ├── replay_attack.wav
    └── ...
```

## 音频规格

- **采样率**: 16kHz
- **格式**: WAV (16-bit PCM)
- **声道**: 单声道
- **时长**: 100ms 片段
- **内容**: 手指敲击声音

## 使用说明

1. 将真实的音频文件放置在相应的用户目录中
2. 确保文件命名符合格式要求
3. 音频质量应足够清晰以供演示使用

## 隐私声明

演示音频文件应当：
- 仅用于技术展示目的
- 不包含可识别个人身份的信息
- 获得适当的使用授权
